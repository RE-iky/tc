# Product Overview

## 无障碍AI教学平台 (Accessible AI Learning Platform)

An accessible AI education platform designed specifically for learners with disabilities, providing barrier-free learning experiences through adaptive interfaces and multi-modal content delivery.

## Core Value Proposition

Helps learners with disabilities overcome information accessibility and tool usability challenges in AI education by providing:
- Accessibility-adapted courses
- Assistive learning support
- Optimized learning pathways

## Target Users

1. **Visual Impairment Users**: Full blind or low vision learners requiring screen reader support, detailed image descriptions, and keyboard-friendly navigation
2. **Hearing Impairment Users**: Deaf or hard-of-hearing learners needing high-quality subtitles, text versions of video content, and AI terminology explanations
3. **Other Disability Types**: Users with physical or cognitive disabilities requiring simplified workflows and clear information hierarchy

## Key Features

- Complete accessibility support (WCAG 2.1 AA compliance)
- Automatic interface adaptation based on user disability type
- Multi-modal content presentation (text, audio, video)
- AI-assisted content description and explanation
- Video subtitle system with AI terminology glossary
- Accessible assignment submission and feedback system

## Design Philosophy

- Accessibility-first design approach
- User-driven interface adaptation
- Multi-modal content equivalence
- Simplified interaction flows
- Clear information architecture
